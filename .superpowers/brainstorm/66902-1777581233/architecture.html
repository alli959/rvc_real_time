Key Design Decisions

Networking Cloudflare Tunnel (no ports exposed, free HTTPS + DDoS protection)
Storage Local SSD for hot data + Cloudflare R2 for backups
GPU RTX 3090 24GB — handles inference + training concurrently
Deployment GitHub Actions → SSH → docker compose pull && up
Cost Control Vast.ai API monitoring + email/webhook alerts at $80/$100
Recovery Auto-restore from R2 on new instance (10-min RTO)
